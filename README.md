# 📊 Customer Churn Analysis | Python & Power BI

## 🚀 Project Overview

This project analyzes customer churn behavior using Python (Pandas, NumPy) and Power BI. It identifies key factors driving churn and provides actionable business insights.

---

## 🔧 Tools & Technologies

* Python (Pandas, NumPy)
* Power BI
* Data Cleaning & ETL
* Exploratory Data Analysis (EDA)

---

## 📊 Key Insights

* Month-to-month contracts show ~50% churn rate
* Customers with high monthly charges are more likely to churn
* Low tenure customers (<12 months) are high-risk
* Fiber optic users show higher churn (~40%)
* ₹160M+ revenue loss identified due to churn

---

## 📈 Dashboard Features

* KPI Cards (Total Customers, Churn %, Revenue Lost)
* Churn Analysis by Contract Type, Tenure, Payment Method
* Customer Risk Segmentation
* Interactive Filters

---

## 📂 Project Structure

* `data/` → Raw & processed datasets
* `notebooks/` → EDA analysis
* `src/` → Data transformation scripts
* `dashboard/` → Power BI dashboard
